<!-- GETTING STARTED -->
## Getting Started

This is an example of how to setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Installation

_Below is an example of how you can instruct your audience on installing and setting up your app._

1. Clone the repo
   ```sh
   git clone https://github.com/estebanfex/vaccinationApp/vaccinationApp
   ```
2. Install pipreqs
   ```sh
   pip install pipreqs
   ```
3. Install all dependencies with pipreqs
   ```sh
   pip install -r requirements.txt
   ```
4. Run the SQL sentences in a database from the files below
   ```sh
   DBvaccines.sql
   DBvacunas.sql
   ```
5. Run the micro web framework flask
   ```sh
   python APIVaxAPP.py
   ```


### Test API with swagger

Use the next file in a swagger editor
   ```sh
   swaggerAPIVax.yaml
   ```


### Unit test

Run the next command
   ```sh
   pytest -q testAPIVaxAPP.py
   ```
